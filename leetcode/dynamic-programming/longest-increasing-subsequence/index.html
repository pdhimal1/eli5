Longest Increasing Subsequence

ðŸ‘¶ The 5-Year-Old Summary

Find the longest chain that keeps growing! [10,9,2,5,3,7,101,18] â†’ [2,3,7,101] or [2,5,7,101] = length 4!

ðŸ§  The Brain Hack

For each element, find the longest chain ending there! dp[i] = 1 + max(dp[j]) for all j < i where nums[j] < nums[i]!

ðŸ’» The Clean Solution

def length_of_lis(nums):
    if not nums: return 0
    
    dp = [1] * len(nums)
    
    for i in range(1, len(nums)):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    
    return max(dp)
