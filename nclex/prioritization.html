ELI5: Who Gets Care First?

Imagine you're a nurse in a busy ER. Four patients need you at once! Who do you help first? Prioritization frameworks give you a logical system to make these decisions.

üöë ABCs - The Golden Rule

Airway ‚Üí Breathing ‚Üí Circulation

Always check these first! A problem with any "A" must be addressed before "B," and so on.

Example:
A patient with: A) Difficulty breathing, B) Chest pain, C) Bleeding cut
Answer: A - Airway problems come first!

üìä Maslow's Hierarchy

When ABCs don't clearly apply, use Maslow. Address lower-level needs first:

  1. Physiological: Food, water, warmth, rest
  2. Safety: Security, safety
  3. Love/Belonging: Intimate relationships, friendship
  4. Esteem: Achievement, respect
  5. Self-Actualization: Fulfilling potential

üîÑ The Nursing Process (ADPIE)

Use this framework for care planning questions:

  1. AAssessment: Collect data, gather information
  2. DDiagnosis: Identify problems (NANDA diagnoses)
  3. PPlanning: Set goals, create care plan
  4. IImplementation: Take action
  5. EEvaluation: Check if it worked

üéØ Prioritization Question Types

"Which patient should the nurse see FIRST?"

  • Use ABCs first
  • Look for signs of respiratory distress
  • Check vital signs

"What is the priority nursing intervention?"

  • Think: What will keep the patient safe?
  • Often, assessment comes before intervention

"Which task can be delegated?"

  • RN can delegate: Assessment, teaching, evaluation
  • UAP (nurse aide) can do: Vital signs, bathing, feeding, toileting
  • LPN can do: Routine meds, wound care, ADLs

‚ö†Ô∏è Common Traps to Avoid

  • Don't jump to interventions: Assess first unless it's an emergency!
  • Don't treat the machine, treat the patient: Look at symptoms, not just numbers
  • Don't assume: Read the question carefully - it might say "except"
  • Don't forget the patient: The correct answer usually prioritizes patient safety
  • Don't overthink: Usually the first thing you'd do in real life is correct
