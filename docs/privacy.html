1. Introduction

ELI5 Park ("we," "our," or "us") is an educational website that explains complex topics in simple terms. This Privacy Policy describes how we collect, use, and protect your information.

2. Information We Collect

We collect NO personal information.

ELI5 Park is a static website that does not:

  • Require user accounts or registration
  • Collect personal information (name, email, etc.)
  • Use cookies or tracking technologies
  • Collect data from children under 13

3. Hosting & Analytics

Our website is hosted on GitHub Pages. GitHub may collect standard server logs including IP addresses and browser information for security and operational purposes. We do not have access to or control over this data.

4. Third-Party Services

We use the following third-party services:

  • GitHub Pages: Website hosting
  • Chart.js (CDN): Used in some interactive demos (Magic Money Tree page)

These services have their own privacy policies.

5. Children's Privacy

Our content is designed to be educational for children and families. We do not knowingly collect any information from children. Parents should supervise children's use of the internet.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page.

7. Contact Us

If you have questions about this Privacy Policy, please contact us through our GitHub repository.
